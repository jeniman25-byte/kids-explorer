# 小探索家 · 微信提审清单（M7）

## 1. 服务器域名白名单
请在微信公众平台配置以下 **request 合法域名**（开发/生产分别填写）：

- `https://your-domain.com`
- `https://api.weixin.qq.com`（若服务端需要回调代理）

路径：`小程序后台 -> 开发 -> 开发管理 -> 开发设置 -> 服务器域名`

## 2. 隐私政策链接
- 页面路径：`/pages/privacy/index`
- 提审表单中的隐私链接填写可访问 H5 链接或小程序内对应说明页截图。

## 3. 功能说明截图（建议）
- 首页录音入口（PAGE 1）
- 生成中进度（PAGE 5）
- 探索页标注（PAGE 6/7/8）
- 特写页（PAGE 9/10/11）
- 图鉴列表（PAGE 12/13/14）
- 多视角四面图/八方图（PAGE A/B）
- 视角标注特写（PAGE C）

## 4. 测试账号信息
- 测试账号昵称
- 测试手机号（如需）
- 测试说明：`首页 -> 录音 -> 生成 -> 探索 -> 特写 -> 多视角`

## 5. 重点合规说明
- 不收集儿童个人敏感信息
- 语音仅用于当次识别与播报
- 生成图片均经过内容安全校验（`imgSecCheck`）
